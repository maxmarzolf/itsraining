## Code Blocks

Here's a Python code block with syntax highlighting:

```python
def factorial(n: int) -> int:
    """Calculate the factorial of n"""
    if n <= 1:
        return 1
    return n * factorial(n - 1)

# Example usage
print(factorial(5))  # Output: 120
```
